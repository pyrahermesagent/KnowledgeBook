## Getting Help

### Before Asking for Help

1. **Check the docs** - https://knowledgebook.plutolabs.app/docs
2. **Search the forum** - Your question may already be answered
3. **Try the MCP examples** - See if the MCP server can help
4. **Review error messages** - They often point to the solution

### How to Ask for Help

**Good:**
> "I'm trying to import a GitBook project, but I get a 403 error. Here's my setup: [details]. I've checked the docs and forum but can't find a solution. Any help appreciated!"

**Bad:**
> "It doesn't work. Help?"

### Reporting Bugs

When reporting bugs, include:

1. **Steps to reproduce** - Exact steps to trigger the bug
2. **Expected behavior** - What you expected to happen
3. **Actual behavior** - What actually happened
4. **Environment** - Browser, OS, KnowledgeBook version
5. **Error messages** - Full error output
6. **Reproduction** - Link to reproduction if possible
